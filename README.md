# University Enterprise resource planning (ERP) system app

## Created using MERN Stack
### Used Technologies 
#### Backend :  
        * NodeJs
        * ExpressJs
        * MongoDB
#### Frontend :
        * ReactJs
        * Redux
        * TailwindCss
        * MaterialUI
#### Safety :
        * Cors
        * Passport-jwt
        * Jsonwebtoken (JWT)
        * Passport

### Getting Started
- First, run the development server:
  -  cd to the backend directory:

```bash
   npm run server
```

- Second, run the client:
    -  cd to the frontend directory:

```bash
   npm run start
```
