# University Enterprise resource planning system app

## Created using MERN Stack
### Used Technologies 
#### Backend :  
        * NodeJs
        * ExpressJs
        * MongoDB
#### Frontend :
        * ReactJs
        * Redux
        * TailwindCss
        * MaterialUI
#### Safety :
        * Cors
        * Passport-jwt
        * Jsonwebtoken (JWT)
        * Passport

## Installation

```
npm install
cd frontend && npm install
```
